# 調查局的上司

* 國安局：實質上層
  * 1949 中國國民黨中央委員會政治行動委員會→1950 總統府機要室資料組→1954 國家安全局
* 法務部：體制上的上層
  * 1938 中國國民黨中央委員會調查統計局→1949 內政部調查局→法務部調查局
* 軍情局：背後的上層
  * 1938 中國國民黨軍事委員會調查統計局→1946 國防部保密局→1955 國防部情報局→1984 \(江南案\)國防部軍事情報局
* 警總：白色恐怖時期曾經的平級單位
  * \(國共內戰\)剿匪總司令部與警備司令部→1949 臺灣省警備總司令部\(保安司令部、衛戍司令部\)→1992\( 結束動員戡亂\)改制為海岸巡防司令部’
  * “郵電檢查、電信監督”\(後歸調查局\)、“特情室\(情報佈建\) ”、“機場、港口安檢”\(後歸警政署\)

# ~1993 國家安全局介紹

1949 情治單位分公不明確，像是對中國敵後工作，調查局也做、國民黨也做、國防部大陸工作處也做，原因：蔣介石喜歡雙軌制統治，兩個“黨”組織\(國民黨與三民主義青年團\)、兩個特務系統\(中統與軍統\)、兩個軍中勢力\(作戰系統與政工系統\)

1950 蔣經國成立“革命行動委員會”，委員會對外化名“總統府機要室資料組”，統合情報單位，委員包括調查局局長、保密局局長、憲兵司令部司令、國防部情報次長、國防部第二廳廳長、台灣省警務處長、台灣省包安司令部副司令。

1952年開始召集國防會議。

1955 年總統府資料組名稱撤銷，成立國防議會\(1967改為國家安全會議\)、國家安全局、動員委員會\(負責調查經濟犯罪及違反國家總動員法的案件\)。

1993 通過國家安全會議組織法、國家安全局組織法法制化，國家安全會議成立。

# 1946 國防部第二廳，亦稱「情報計畫廳」

* 第一處：主管行政及計畫
* 第二處：主管邊疆情報

* 第六處：主管保密防諜。下分兩個科：①防盜竊機要檔案；②偵察共產黨地下工作。共15人左右，均系軍統分子，在各廳、局普遍設有保密防諜小組。全廳內勤人員共約300人，全國各地的站、組及掌握適用的人無法計數。該廳1946年秋曾採用過「七級編制」，分廳、署、司、處、科、組、股\(層次順序記不真切\)，人員達500餘名。侯騰代廳長時，即按上述編制改組。

# ~1946 國防部第二廳技術實驗室

* 負責蒐集國內外軍事情報，並進行整理、分析、判斷等，為國防作戰計畫的制定提供條件，並計畫布置保密防諜等活動，自成一特務系統，與軍統特務組織聯繫配合，又獨樹一幟，相互間明爭暗鬥。
* 國防部第二廳技術實驗室所在位置為現今新店大豐國小，在此訓練情報照相、縮影、底片沖洗、藥水配方、情報相機使用；
  * 1955年3月14日，國防部第二廳技術實驗室改為「國防部技術研究室」。
  * 1963年3月1日，技術研究室改組為「國防部電訊發展室」。
* 期間總統府資料室\(國安局前身\)也成立類似實驗室，包括照相、化學、爆破、毒藥等部門。

# 1947 國防部第二廳-情報學校

1947年該廳在南京珠江路小營辦有「情報學校」一所，由陳誠任校長，侯騰主持，教務主任王子清代侯負實際責任。分：①普通班，專訓練國內情報特務；②高級班，訓練高級情報特務；③外事班，訓練武官及通譯。1948年停辦。

該廳同各行轅\(綏署、剿總、軍政長官公署等\)、各綏區的第二處，以及部隊軍事情報處、科在業務上層層聯繫，人事上也多方操縱，建立一個所謂「情報參謀系統」。各第二處的負責人多由該廳選拔或推薦，部隊情報參謀多系該廳情報學校或前軍令部第二廳所主辦情報參謀訓練班畢業的，從而形成了一支獨立的特務系統。

# 1949 國防部第二廳-特務武裝部隊



* 1949 年該廳策劃布置地下特務武裝部隊，企圖侵入中共控制區或留在當地等待解放軍占領後進行反共的游擊戰，以破壞中共政權，建立國民政府政權並建立游擊根據地長期存在發展，等待三次世界大戰爆發，配合反攻，以謀復辟興。
  * 布置的區域有中共「老解放區」，「新解放」區及即將「解放」的地區。
  * 所利用的對象包含有各地封建把頭、幫會、道會門、土匪、流氓、惡霸、土頑、軍閥、政客、閒散軍官、省縣保全部隊、軍、師、團管區等等。
  * 組建名義繁多，有「游擊縱隊」、「突擊隊」、「挺進隊」、「綏靖軍」、「反共救國軍」、「反共自衛軍」等。
* 1958年改為國防部特種軍事情報室。
* 1985年7月1日，與國防部情報局合併成「軍事情報局」。

# 1950 國防部政治部幹部訓練班(石牌班)

調查局之父沈之岳主持，年輕時作為特務潛伏在延安抗日軍政大學，一度當過毛澤東的機要祕書，班主任為蔣經國。

調查局、保密局、保安司令部、憲兵、警察、國民黨中央二六組的中高級成員都要去受訓。

石牌班為在台灣首次公開對外招考的地下人員訓練班，於現今石牌國小處受訓；於現今台北榮民總醫院處進行野外求生訓練。

訓練目的：情報蒐集、爆破、策反、游擊、密寫、武器使用、體能訓練、武術、野地求生、作戰技巧；思想教育包括革命理論、匪黨理論批判、加強仇匪與恨匪觀念，歷時半年。

同期九十三名成員，除了八人以外全送到敵後，最後只剩兩人活著，其中一人受了二十年勞改，另一名成功建立組織後返台就拒絕再赴任。

1953 設置高級班，調訓各情治機關中級幹部，稱作”聯訓班”

石牌班在安全局成立後繼續存在，訓練地點改到台北市劍潭青年活動中心，之後再搬到淡水鎮的海邊沙崙里，最後搬到花蓮，和美國合作專門訓練特種人員。

# 1950 敵後工作

* 民國四十年代後期，有三個單位負責敵後工作：
  * 國防部情報局、國民黨中央委員會第二組和國防部特勤室。
  * 在日本、韓國、泰國邊境、香港設立“海外基地”，尤以香港最多；日本則有調查局第一處配置五人的日本工作組監控台獨、華僑；國安局則是配置東京督導組。
    * 日本工作組以“橫濱中華學校校長”為名掩護，最後全部被公開。
  * 香港的前進基地工作重點，負責訓練香港地區的青年到大陸去為我們建立組織、蒐集情報、從事敵後破壞工作。
* 派到大陸的人有兩種來源，第一種台灣受訓並透過祕密管道潛入大陸；另一種是在“海外基地”找當地居民或旅客，吸收後以自然身份進入。
* 訓練課程為如何蒐集情報、傳遞情報、判別何種情報是有用或無用、密寫通訊等技術。
* 以可觀經費「組織建立費」、「據點建立費」吸收人員，到中國後以「家書」寄到指定通訊地點報平安代表據點成立，只要見到家書代表情報員在大陸的據點已經成立，就可以，把成果呈報給中央領到工作經費。
* 有時他們會託人送一張照片，比如在黃花崗上貼反對共產黨字樣的標語照片，獎金更高；調查員為了績效，就算識破了也不會說；國內上級又何嘗不知道？但也不敢說。
* 1955~1965 年，調查局在第一處副處長指導下於香港進駐祕密組織。

# 1963 (傳)情報局試圖刺殺當時中華人民共和國主席劉少奇

1954年萬Ｘ計畫，情報局在泰寮成立反共游擊隊。

1963年4月20日，沈之岳時任情報局副局長兼中二組副主任，化名孫子超，4月20日秘密潛入葡屬澳門，試圖刺殺當時中華人民共和國主席劉少奇。

沈在澳門新新酒店、同盟酒店各開一個房間，實際上住在「中二組」澳門特一組的駐地；沈行蹤被共產黨完全掌握，華南辦事處將資料直送北京。

中共高層曾考慮將沈之岳從澳門捉拿回大陸歸案，最後採取較緩和的做法，由公安部通過外交途徑通知澳葡政府。澳葡政府逮捕了一批特務，將沈驅逐回台灣，沈因此受到撤職查辦的處分。

1963年5月軍情局幹員在柬埔寨金邊刺殺前中國國家主席劉少奇行動失利，相關人員遭到逮捕。

# 1965 安全局：馬拉加西(馬達加斯加)大使綁架回台事件

民國五十年代我國邦交國很多，但政府很擔心中共會把台灣邦交國一個一個拿走，也怕駐外史官變節投靠中國。

安全局接到法國情報，說我國駐馬拉加西大使有和中國通信，後來情報更新為打算投共。

國安局派特勤組\(醫官、報務員、三個行動員\+陳震上校組長\)包民船\(客貨兩用輪船\)綁架可能投共的汪公紀大使，受命如果大使不從可以當場實施“制裁”。

民船抵達後以首航酒會作為幌子誘騙大使，大使上船後國安局副局長黃德美給汪公紀看外交部令函並扣押回台。

事後安全局因無法證實法國情報的真假使案子最後沒有成立，汪公紀又回到外交部上班，外交部重新派人出任大使。

# 1949~1991 白色恐怖



* 始於1949年5月20日臺灣省政府及臺灣警備總司令部對臺灣（臺灣省）全境實施戒嚴，至1991年6月3日《戡亂時期檢肅匪諜條例》廢止為止。
* 白色由來 1
  * 1870年，普法戰爭的爆發，法蘭西第二帝國（1852年\-1870年）的拿破崙三世被俘，法蘭西第三共和國（1870年\-1940年）宣佈成立，並相繼鎮壓了巴黎人民。
  * 1871年3月28日，身為「工人階級實質上政府」的「巴黎公社」正式成立，而法國政府軍在普魯士軍隊的幫助下，對巴黎公社成員發動攻擊。公社成員也群起反抗。因為一時找不到代表公社的旗幟，公社裡的一名女工於是從自己身上的紅裙，撕下一塊紅布，作為公社的標誌。從此以後，紅色便引用為一切進步熱情、反抗不義的階級解放之符號。與此相對，代表反動、保守的勢力，便是『白色』，而由其發動的一切恐怖鎮壓行動，就是所謂『白色恐怖』」。
* 白色由來 2
  * 法國大革命當時以白色為代表色的右派波旁王室對左派雅克賓黨人所採取的報復行動。
* 白色由來 3
  * 在帝俄時代，白色是沙俄帝國皇室的代表顏色，沙皇常被冠以「白色沙皇」之稱，白色並無任何貶義色彩。在西元1917年後的俄國內戰中，反共的俄國「白軍」與俄共布爾什維克的「紅軍」相對，遂使用「白色恐怖」一詞來形容白軍的殘暴。
* 中國共產黨官方觀點使用「白色恐怖」一詞來形容由軍統和中統執行的打擊中共勢力的隱密行動，包括暗殺、暴力鎮壓及逮捕共產黨員等，中共亦將這些中華民國政府控制區域稱為「白區」，將共產黨控制區域稱為「紅區」。

# 調查局介紹

沈之岳：“我們就是領袖的耳目，我們就是領袖的狗\!”

早期調查局位於基隆路時，調查局屬於國民黨台北市黨部第十六區黨部，調查局長兼任區黨部常委；後調查局搬遷至新店中華路、但黨部仍在台北。

調查局在國民黨內的編制屬於“台北市黨部第十六區黨部”，早年調查局局長兼任區黨部常委、中央常委，區黨部辦公室直接設在調查局內。

調查局是國家領袖的核心幹部，相對而言黨員對調查局沒有份量。

現在調查局黨部辦公室設在新店中華路調查局本部對面“崇光大樓”頂樓，因為不想被民宅查看，以個人名義花錢買下再捐給調查局使用。

調查局實際運作時與國民黨關係不深，過去有局長為競選中央委員時常利用調查局資源與金錢

# 1961 人事室第二辦公室(人二室)

保防業務：調查局保防班受訓後派駐到各機關，設立的單位叫做安全室或保防室。

改制前，安全室人員由保密局“推薦”再經單位提名同意，後來成為調查局安插人員的市場。

調查局第二處主管工作為保防、各行政機關也會派駐執行保防工作人員、都是由調查局招訓產生。主要受訓班別為「查核班」\(一般調查員為「調查班」\)，查核班升遷不易且難以調職或轉為調查班工作。

沈之岳祕密調查行政院秘書長費驊“匪黨分子”，遭費驊發現，費驊趁沈之岳出國時下令調查安全室制度，提報蔣經國，蔣經國主持國內治安會議「永靖會議」時指出因為在安全單位負責保防的人事查核單位都在作威作福，裁示裁撤、併入機關人事單位中。

換湯不換藥，改為人事室第二辦公室，原主任職稱改為機關內其他一級主管的職銜，如委員、參事、監督人等等

台北市政府因為人室單位編制龐大，下設六科，包防業務被分配到第五、第六科，被外界笑為“人五人六”。

原本人二室曾經打算取名做“人查室”，但太容易被罵做人渣室，所以又改回人二室。

# 調查局工作：布建

1949 國民黨負責滲透單位有中統、軍統、憲統、警總保安處與國民黨中央第六組。

收集情報的要領類似新聞寫作，主要注意人、事、時、地、物、為何、如何等等，調查局在得到情報後會再交給另外一名布建人員去複查。

負責單位為調查局第五處\(改制前為第一處\)，民國七十年調查局設在全國的步建人員約有三萬，當時調查局長阮成章希望提高到“五百人中就有一個”

調查局外勤人員一千四百多人中有五百人負責跑情報，而據傳每個人奉命吸收二十名佈建人員。

線民獎金預算稱之為“諮詢費”，為了避免線民申報稅金時身份曝光，使用“分離申報”制度；請領過成為匿名，所以調查員也可能冒名請領。

情報蒐集後交由局本部第一處整理，最重要的情報彙整後再往上報給局長、副局長、國安局業務處和總統。

與行政院或相關部會有關的，會篩選至只剩他們自己的業務後給部會首長。

* 布建方式
  * 打入：發現某人或團體可疑時就派人去接近他們，然後順勢讓對方吸收變成我們的內應，通常稱為臥底。
    * 民國四十九年翁文維擔任臺北縣站站主任，涉及風紀案依照“家規”關入誠舍，關一年後出獄連連高升直到局長才退休。
  * 拉出：從目標對象的群體中選擇可以為我們所使用的人選並加以運用，或是以金錢方式利誘、收買，請他們提供情報給我們，調查局主要以使用拉出為主。
* 布建可以分為「一般布建」、「重點布建」、「偵破布建」三種，調查員一般要掌握3~40人布建人員，以當時調查員約有2000多人的編制，其他單位如憲兵調查組、情報局、警總也都有自己的布建人員。
* 一般布建：在各地區普遍設置「通信員」，平時不支薪，有特殊表現時才會敘獎，但在一般三節時也會送些小禮物。
* 重點布建：針對特殊團體或政治組織所作的布建，新聞媒體內也會有，會按月支新、用化名簽收條。
* 偵破布建又稱為專案布建，針對特定重大販毒、販槍或經濟犯罪組織，鎖定偵查對象以後以打入或拉出方式佈眼線進去，有時候室偵查目標的親友、鄰居或是任何可能接觸到目標的人，而且一次至少布兩個互不相干的人。每個月可以之領到的薪水會相當高。
* 海外佈建：大多由史館內的安全調查官來進行主管及指揮。
* 其他方法：假扮一起被關的犯人難友突破心防，但非常態。

# 情報單位工作：監聽

* 1999 公佈“通訊保障及監察法”正式讓監聽過程法制化，但長期以來都有監聽歷史。
* 警總時期電信總局有一個特殊機房，專門監聽警總等特務單位交代的電話號碼，當時只有有線電話，訊號必須在電信局的交換機上傳遞。監聽分A、B、C三級。
  * C級是二十四小時掛在錄音帶上，等到需要時才調出通聯紀錄，以供查核，一般政府官員、民意代表屬於這類；
  * B級是電話掛在錄音機上，二十四小時整理一次紀錄，這是針對「三合一敵人」（赤色分子、黨外分子、臺獨分子）等人士及列管的政府官員與外國人士；
  * A級則是像時常針砭國民黨威權統治的重量級異議分子－如郭雨新這類二十四小時跟監的重點人士，只要他家電話一響，都會自動先接到電信局監聽單位，從哪裡打都一清二楚，日後康寧祥的住家及研究室都受到「A級待遇」。
* 現在有公開證實具有監聽設備的兩個單位分別是調查局、刑事警察局。
  * 調查局監聽一般的有線電話與中華電信的行動電話。
  * 刑事警察局全數監聽民營行動電話。
* 擁有監聽設備的其他單位：國家安全局、軍事情報局、海巡署情報處以及國防部直屬的幾個特殊單位\(電信發展室\)。

# 1949

局長季源溥帶領調查局人員從由南京經過廈門到廣州，再來到台灣，還沒找到房子就病倒了；調查局副局長張益民則是帶人員與官章，到重慶成都，見情勢不對丟下印信不辭而別飛到香港。

成都的調查局本部職員有一百多人，但沒有領導人、沒有制度負責人、也沒有經費；此狀況下CC系頭子陳立夫\(創辦中統的元老\)口頭通知張慶恩暫代調查局長職務後，由張慶恩負責調查局撤退到台灣的行動。

從成都逃來，除了公文什麼都不能帶，人人衣物缺乏、經費支絀，沒住所也沒辦公室。

最初在台北市雙城街找了棟二十多坪的日式房屋當局本部，一開始只有十幾個人在支撐危局，後續中國各省的老職員輾轉來台，才轉到環河南街做局本部。

1958 季源溥因與國安局長鄭介民交惡，後被調為內政部次長，由張慶恩\(最後一個屬於中統的局長\)繼任。

# 1955~

* 1955 蔣經國\(軍統系出身\)為了消滅派系鬥爭，要求調查局把對中國敵後滲透工作移交給情報局，並實施兩局\(中統調查局、軍統情報局\)人事交流，幾種主張
  * 把工作能力最弱的人流到對方去
  * 把最擅長搞人事糾紛的，“過局”去搗亂。
  * 挑選工作能力最強又善於奪權的過去，長況對方實權。
  * 挑老的過去等待退休。
  * 雙方派系為了消滅彼此，互相指控告密。
* 以調查站為例，當情報局人員流入調查局初期，先是讓原保密局各站暫時與調查站並存，然後逐漸裁併為一；例如調查局台北實驗區\(後來的台北市調查站\)，與保密局台北站同時存在，直到 1955 年江南案後保密局改稱為情報局，調查局改隸屬司法行政部，兩個站才告統一，之後站主任與副主任總不免是：一中統、一軍統。經常會互相密告彼此。
* 1956 調查局開始接掌台獨案件，當時對海外台獨人事處理原則為“政治問題政治解決，不受逮捕法辦”
* 1958 密告文化
  * 季源溥在失去蔣經國信任後調為內政部政務次長，當時內政部是一個空殼部，人員少經費少權力更小，明升暗降；原副局長張慶恩繼任。
  * 同年“出版法”在立院引發軒然大波，主要是 CC 系立委反對，認為只有扼殺出版事業，立法院 CC 派系與情報局互相私下密告。

# 1967 立法禁止刑求

1965\.4 立法院質詢是否有十二小時疲乏刑求，而調查局保密作風沒人知道；而當時只有刑案有獎金，所以刑求作風擴大。

1965 沈之岳偵辦黃豆弊案、逮捕數名立監委員，以他們在立院的發言紀錄作為“犯罪證據”。

1967年3月3日依刑事訴訟法規定解送指定之看守所，不得寄押於調查局之留質室。

雖有立法，但此時調查局仍不受法律管轄、違法也不需送司法機關，直至1989 吳東明局長才開始強力要求守法。

# 1984 301專案

1984年3月1日許多大專院校發現鼓吹罷課與學運的傳單。

調查局動員所有在學生中的布建都沒有訊息，後請調查局第六處\(負責科技工作\)進行比對，找出複印位置的特殊痕跡、再找進口商比對銷貨紀錄

將台北地區所有銷售地點標示於地圖上，將地圖畫格逐一調查比對，調查方式為請影印店老闆幫忙，假扮保養維修影印機的技師免費替店家保養、檢查影印機，偷印文件回來比對特徵。

再派調查員去補習班假裝上課、打聽消息觀察狀況，比對為台大政治系學生，並被調查局懷疑與他們監控的其他學生劉一德、賴勁麟、李文忠有關而進行約談，

# 1990 港台洗錢案

1993年1月5日，台北市調查處中正站副主任王任謙與4名調查官，因王升官派任俄國\(後續立委說是慶祝祕書顧取隆升任澎湖站主任、站主任李錨勝任東機組主任乃政治攻防牽強附會\)慶功送行，之後轉往金殿夜總會續攤並找來柏青哥店負責人陳金盈，席間商人方敦泄\(自稱是方椲柏\)、陳的李姓女祕書陸續到場。次日凌晨，方敦泄在包廂性侵酒醉李女，雖被服務生撞見，但王任謙卻未阻止方，讓方性侵得逞，事後王任謙到賓館開房間把受害者帶入後離去。

# 1994 追緝海外逃犯

追緝海外逃犯需要跨單位共同協調，例如當察知某個罪犯逃亡國外後，必須先由外交部領事局註銷此人的護照並通知當地國，當地國再以“沒有有效的國外旅遊證件”把這名罪犯緝捕。

人犯落網後，再由警方或調查局等司法警察執行解送歸案。

調查局有機會跟國外司法單位開會，都會推廣“以驅除出境代替引渡”，因為罪犯一旦驅逐出境，就會被遣反回台灣，而若遣返的飛機是我國籍的飛機，我們的執法人員就可以在飛機上直接執法，逮捕犯人，因為照法律規定，我國國籍的航空器就是我國領土的延伸。

若班機不是我國國籍，也可以一下飛機踏入中華民國的領土就能被逮捕，這都比冗長的引渡程序快的多。

# 1994 調查局逃漏稅案

調查局本身負責追查逃漏稅，過去也會針對異議人士調查逃稅。

過去調查局人員是由會計室統一申報最低級距6%所得稅，所以計算夫妻所得時只有妻子收入、所用稅率會更低，且預扣通常是少扣。

調查局長當時每年短報四萬所得稅。

事件報導後，媒體追問財政部，財政部承認像調查局這種敏感單位沒必要是不會去查稅的。

後續調查局取消統一申報，並且統計五年來靠繳憑單重新申報。

# 2001 國安密帳之鞏案

2001 年媒體報導李登輝非法挪用三十五億，而這則新聞是因為媒體掌握大大批國安局流出的祕密文件，而這些文件保管的正是劉冠軍，調查局開始調查。

翻閱劉冠軍簽辦的文件時，發現有個文件的承辦人電話分機號碼是四位數自，但依照文件上的日期推算，當時安全局內部公務電話系統尚未更新，分機號碼應該都是三碼，因此確定這份文件是偽造的。

# 2007 修正調查局組織條例

1956 司法行政部調查局組織條例，調查局下設一至七處，並設有研究、設計、訓練三個委員會與人事、主計、機要三個室，除了明定第一至七處所掌理的事務外，條例第二條還規定“司法行政部調查局，掌理有關危害國家安全與違反國家利益之調查保防事項。前項調查保防之事項，由行政院定之。”

由此行政院特頒調查局業務執掌，分別是：內亂事項、外患事項、洩漏國家機密事項、妨害國家總動員事項、貪汙瀆職事項、肅清煙毒事項、妨害國幣事項、妨害戰時交通電業維護事項、完反電信管理事項、漏稅查緝事項、上級機關特交之事項。之後又把洗錢防治業務也加到調查局工作中。

因應這些業務執掌，調查局成立非常多任務編組，如經濟犯罪防制中心、肅貪處\(後更名為廉政處\)、緝毒中心、洗錢防制中心等，另還有功能編組如海外室、外事室、資訊室、聯絡室等。

但這些任務編組或功能編組在組織條例上都不存在，所以也不會有正式的職缺，因此挪用職缺是調查局長期以來嚴重的問題。

且這些調查保防事項都是行政院的行政命令訂頒，行政院基於各種原因隨時可能收回業務。

---

逢重大政治情勢變化或者上級長官好惡改變，就會取消調查局的某些執掌，例如成立廉政署取代調查局廉政處、成立緝毒署取代調查局緝毒中心，連洗錢防治法修法時，法務部都不同意將行政院指定的銀行申報機關正式修正為調查局等等。

2007 11/14葉茂盛指示修訂組織條例，並且跳過行政體系直接由立院切入，劉猜測局長事前已經想好一整套策略，包含指示人事室草擬修正條文、私下找可信賴的學者專家研商規劃組織條例修正草案，再透過國會組向立法院各黨團遊說，並且動員所有外勤處站主任緊盯轄區立委說服，等建立共事後才在立院發動奇襲。

正常程序下，調查局組織條例若要修正需經由國安局、法務部、行政院點頭，行政院最後綜合整體意見並於審查結束後才能召開院會討論修法草案，全案通過後再送進立院進行一二三讀。

2007 11/20 國民黨、民進黨、親民黨、台聯黨和無黨籍聯盟共同提案“法務部調查局組織條例修正草案”

2007 11/23 院會中超過二十位委員連署逕赴二讀。

行政院、國安局果然跳腳，但朝野五黨立委共同提案，行政院難以阻撓，在通過逕赴二讀決議後行政院召集國安局、考試院、法務部等單位召開政策協調會。

# 總攬

![](img/03_241.webp)

~1950：偵防為主、包含海外行動，調查員只有在中國時期的國民黨黨員，毫無法律概念。

1967：修法起訴需要檢察官、禁止私刑，調查局成立招待所規避。

1980：美麗島大審、兩岸開放，調查局增加重心到犯防與防治金融犯罪。

1989：江南案，偵防工作交由調查局負責。

1991：獨台會案，刑100修法，叛亂罪需有行動而不能只有思想與宣傳。

2000：美麗島系當選總統副總統。

2008：馬英九當選總統，開始簽訂 ECFA ，調查局與中國交流。

# 1940 馬場町刑場

![](img/03_247.webp)

日本時代，日本陸軍於明治42（1909）年設立「臺北練兵場」

馬場町刑場為民國30（1940）年代末期至40（1950）年代初期，槍決政治案件當事人的主要刑場。

在此處遭受槍決的政治案件當事人，包括38（1949）年山東學生流亡案（又稱澎湖七一三事件）張敏之等人、省工委案件中的基隆中學案鍾浩東、臺北市工作委員會郭琇琮案許強等人、台南市委會郵電支部案的施水環與丁窈窕、綠島再叛亂案13人、蘆洲李宅李蒼降。

共諜吳石被中國旅客獻花。

# 1951 水源地刑場

1951年4月後，水源地一帶逐漸取代馬場町成為主要刑場，一開始主要在螢橋一帶，5月起開始往新店溪上游遷至當時國防醫學院附近。

後有農民向省參議會提出陳情，表示因為水源路六巷土地常被用作刑場執行槍決，不敢前往該地耕作。

1952年2月川端橋南端刑場取代水源地一帶成為主要刑場。

# 1950 六張犁亂葬崗

1950 年代遭槍決之政治犯，會於臺北車站張貼公告，通知家屬三天內前來認領，其遺體統一交由極樂殯儀館浸泡福馬林進行防腐處理，家屬必須付出可觀的「贖屍金」、「子彈錢」才能領回遺體。

無人認回的受難者遺體，部分送到國防醫學院供解剖教學後進行火化或土葬，其他則由極樂殯儀館送至極樂公墓（六張犁墓地）進行掩埋。

旁邊有白崇禧墓園、蔣渭水墓園。
